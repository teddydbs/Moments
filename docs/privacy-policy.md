# Politique de confidentialité - Moments

**Dernière mise à jour : 12 décembre 2025**

Teddy Dubois ("nous", "notre", "nos") exploite l'application mobile Moments (ci-après dénommée le "Service").

Cette page vous informe de nos politiques en matière de collecte, d'utilisation et de divulgation des données personnelles lorsque vous utilisez notre Service, ainsi que des choix que vous avez associés à ces données.

## 1. Informations collectées

### 1.1 Informations que vous nous fournissez

Lorsque vous utilisez Moments, nous collectons les informations suivantes :

**Informations de compte** :
- Adresse e-mail (pour l'authentification)
- Nom complet (fourni lors de l'inscription)
- Photo de profil (optionnel)
- Date de naissance (optionnel)
- Numéro de téléphone (optionnel)
- Adresse postale (optionnel)

**Informations d'authentification** :
- Si vous utilisez Google Sign In : votre adresse e-mail Google et votre nom
- Si vous utilisez Apple Sign In : votre identifiant Apple et votre nom (si partagé)

**Contenu utilisateur** :
- Événements que vous créez (titre, date, description, lieu, etc.)
- Invitations que vous envoyez
- Listes de cadeaux (wishlist) que vous créez
- Photos d'événements que vous uploadez

### 1.2 Informations collectées automatiquement

Nous collectons automatiquement certaines informations techniques :
- Type d'appareil et modèle
- Version du système d'exploitation (iOS)
- Identifiants uniques de l'appareil
- Logs d'erreurs et données de diagnostic

### 1.3 Informations non collectées

Nous **NE COLLECTONS PAS** :
- Votre localisation GPS en temps réel
- Vos contacts (sauf si vous les ajoutez manuellement à vos événements)
- Vos messages privés (l'app n'a pas de messagerie)
- Vos données de paiement (aucune transaction financière dans l'app)

## 2. Utilisation des données

Nous utilisons vos données pour :

**Fournir le Service** :
- Créer et gérer votre compte
- Vous permettre d'organiser des événements
- Envoyer des invitations à vos événements
- Synchroniser vos données entre vos appareils
- Afficher votre profil aux autres utilisateurs (si vous partagez un événement)

**Améliorer le Service** :
- Analyser l'utilisation de l'app pour améliorer les fonctionnalités
- Corriger les bugs et problèmes techniques
- Développer de nouvelles fonctionnalités

**Communications** :
- Vous envoyer des notifications concernant vos événements
- Vous informer de mises à jour importantes du Service
- Répondre à vos demandes de support

## 3. Base légale du traitement (RGPD)

Si vous êtes dans l'Espace Économique Européen (EEE), notre base légale pour collecter et utiliser vos données est :

- **Exécution d'un contrat** : Nous avons besoin de vos données pour vous fournir le Service
- **Consentement** : Pour certaines utilisations (comme les notifications), nous demandons votre consentement explicite
- **Intérêts légitimes** : Pour améliorer notre Service et assurer la sécurité

## 4. Partage des données

### 4.1 Avec qui partageons-nous vos données ?

**Fournisseurs de services** :
- **Supabase** (hébergement de la base de données) - États-Unis
- **Apple** (si vous utilisez Sign in with Apple) - États-Unis
- **Google** (si vous utilisez Google Sign In) - États-Unis

**Autres utilisateurs de Moments** :
- Votre nom et photo de profil sont visibles par les personnes que vous invitez à vos événements
- Les informations d'événements que vous créez sont visibles par les invités

### 4.2 Nous ne vendons jamais vos données

Nous **NE VENDONS JAMAIS** vos données personnelles à des tiers.

### 4.3 Divulgations légales

Nous pouvons divulguer vos données si la loi l'exige ou pour :
- Respecter une obligation légale
- Protéger nos droits ou notre propriété
- Prévenir une fraude ou un abus
- Protéger la sécurité des utilisateurs

## 5. Stockage et sécurité des données

### 5.1 Où sont stockées vos données ?

Vos données sont stockées sur les serveurs de **Supabase**, situés principalement aux **États-Unis**.

### 5.2 Combien de temps conservons-nous vos données ?

- **Données de compte** : Tant que votre compte est actif
- **Événements et contenu** : Tant que vous ne les supprimez pas
- **Données de diagnostic** : Maximum 90 jours

Lorsque vous supprimez votre compte, toutes vos données sont **définitivement supprimées** sous 30 jours.

### 5.3 Sécurité

Nous mettons en œuvre des mesures de sécurité pour protéger vos données :
- Chiffrement HTTPS pour toutes les communications
- Authentification sécurisée (OAuth 2.0)
- Row Level Security (RLS) sur notre base de données
- Stockage sécurisé des clés API

Cependant, aucune méthode de transmission ou de stockage électronique n'est 100% sécurisée.

## 6. Vos droits

### 6.1 Droits RGPD (utilisateurs EEE)

Si vous êtes dans l'EEE, vous avez les droits suivants :

- **Droit d'accès** : Obtenir une copie de vos données
- **Droit de rectification** : Corriger des données inexactes
- **Droit à l'effacement** : Supprimer vos données ("droit à l'oubli")
- **Droit à la portabilité** : Recevoir vos données dans un format structuré
- **Droit d'opposition** : Vous opposer à certains traitements
- **Droit de limitation** : Limiter le traitement de vos données

Pour exercer ces droits, contactez-nous à : **teddydubois45@gmail.com**

### 6.2 Utilisateurs californiens (CCPA)

Si vous êtes en Californie, vous avez le droit de :
- Savoir quelles données nous collectons
- Demander la suppression de vos données
- Refuser la vente de vos données (nous ne vendons pas vos données)

### 6.3 Comment exercer vos droits ?

**Dans l'app** :
- Allez dans **Paramètres** → **Compte**
- Modifiez vos informations personnelles
- Supprimez votre compte (suppression définitive)

**Par e-mail** :
Envoyez un e-mail à **teddydubois45@gmail.com** avec votre demande.

Nous répondrons sous **30 jours**.

## 7. Données des mineurs

Moments est destiné aux utilisateurs âgés de **13 ans et plus**.

Nous ne collectons pas sciemment de données d'enfants de moins de 13 ans. Si vous êtes un parent et que vous découvrez que votre enfant nous a fourni des données, contactez-nous immédiatement.

## 8. Transferts internationaux de données

Vos données peuvent être transférées et stockées sur des serveurs situés en dehors de votre pays de résidence, notamment aux **États-Unis**.

Si vous êtes dans l'EEE, nous nous assurons que ces transferts respectent le RGPD en utilisant :
- Les clauses contractuelles types de la Commission européenne
- Le Privacy Shield Framework (si applicable)
- Les garanties appropriées pour protéger vos données

## 9. Cookies et technologies de suivi

Moments **N'UTILISE PAS** de cookies de suivi publicitaire.

Nous utilisons uniquement :
- **Cookies de session** : Pour vous maintenir connecté
- **Stockage local** : Pour mettre en cache vos données et améliorer les performances

Vous pouvez désactiver ces fonctionnalités dans les paramètres de votre appareil, mais cela peut affecter le fonctionnement de l'app.

## 10. Liens vers d'autres sites

Notre Service peut contenir des liens vers des sites externes (par exemple, liens de wishlist vers Amazon).

Nous ne sommes **PAS responsables** du contenu ou des pratiques de confidentialité de ces sites. Nous vous encourageons à lire leur politique de confidentialité.

## 11. Modifications de cette politique

Nous vous notifierons de tout changement important :
- Via une notification dans l'app
- Par e-mail (si vous avez fourni votre adresse e-mail)
- En mettant à jour la date "Dernière mise à jour" en haut de cette page

Nous vous encourageons à consulter régulièrement cette page.
